OpenProject Timesheet Import

Import OpenProject time entry CSV file to Odoo.

Go to your OpenProject instance's /time_entries, filter the necessary time entries (eg. for last month) in the Details tab and click on the CSV link at the bottom to download the CSV file.

In Odoo, go to Timesheets → To Approve → OpenProject Timesheet Import, select the downloaded CSV file and set the Date Format which is used for dates in the CSV file (this depends on your settings in OpenProject) and/or other settings. When done, click the Upload CSV button.

In the Employees tab map OpenProject users to employees in Odoo.

In the Projects tab map OpenProject projects to projects in Odoo. When done, click on the Import button.
